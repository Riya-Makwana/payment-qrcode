#  UPI Payment QR Code Generator using Python

This project generates **UPI payment QR codes** using Python.  
The QR codes can be scanned using **PhonePe, Paytm, Google Pay**, or any UPI-supported app.

---

##  Features
- Takes **UPI ID as user input**
- Generates UPI payment links
- Creates QR codes for:
  - PhonePe
  - Paytm
  - Google Pay
- Saves QR codes as image files
- Displays QR codes automatically

---

##  Technologies Used
- Python
- qrcode library
- Pillow (PIL)
